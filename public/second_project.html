In-Vehicle Infotainment for Four-Wheeler

Description

  • This project involves the development of a comprehensive frontend interface for a car infotainment system with integrated HVAC control. The frontend interface will offer basic functionalities such as multimedia playback, navigation, and HVAC adjustments, designed to enhance the in-car user experience.

    Roles and Responibilities

    • Developing Complete UI Elements using Figma
    • Developed Ui elements of IVI such as Radio Screen, Date and Time setting, Unit, Volume Setting and Location Sync Screens
